# Reading_notes
* 读书笔记摘录

#### 哲学与思想
* 《中国哲学简史》冯友兰。
* 初次阅读还是大学之时，收获颇丰，对我们中国哲学的发展脉络和内核产生了新的认识。后又陆续读过几次，每次阅读都有新的收获。
* 个人觉得非常适合对哲学/思想感兴趣的入门书。在有基本框架之后，对理解各派各人具体思想很有帮助。
* 其中对于人的"超现实需求"、东西方哲学发展差异、中国哲学的发展脉络等的论述，很是精妙。
* 《老子今注今译》陈鼓应。陈鼓应的这一版对道德经的注释比现今流通的很多版本水平高很多，注释详尽准确，很多解释通俗易懂，个人非常喜欢。
* 《菜根谭》洪应明。高度凝练、优美的格言集。
* 《人生的智慧》叔本华。此书名看起来像是现今的鸡汤书一般，其内容却是深具洞见。
